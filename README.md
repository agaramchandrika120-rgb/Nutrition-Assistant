# Nutrition Assistant

## Project Overview
Nutrition Assistant is a full-stack MERN application that helps nutritionists manage clients, create meal plans, and track client progress.

## Features
- User Registration
- User Login
- Client Management
- Meal Plan Management
- Progress Tracking
- MongoDB Database Integration
- REST API using Express.js

## Technologies Used

### Frontend
- React
- Vite
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt

## Project Structure

Nutrition-Assistant/
├── frontend/
├── backend/

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
